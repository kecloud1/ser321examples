You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=1&num2=2
  • /github?query=users/[USERNAME]/repos

    -- returns a list of each repository belonging to the specified user

  • /story?name=[STRING]&pronoun=[STRING]

    -- returns a story about the person using the personal pronoun entered. If no pronoun is entered a gender-neutral one is used. If no name is included, you will get a story about Bob. Example pronouns: he, she, they

  • /feedthedog?dogID=[INTEGER]&numTreats=[INTEGER]

    -- Feeds the requested dog the number of treats specified and displays result. The dogID must be a number between 1 and 5.
